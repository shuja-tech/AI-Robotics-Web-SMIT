# 🚀 Techno Kids AI & Robotics - Phase 1 Curriculum

This platform is a highly polished, interactive single-page application designed for the **Saylani Mass IT Training (SMIT) Program**. It elegantly showcases the "Phase 1 Curriculum", a revolutionary hardware and logic training track designed to ignite curiosity and build a robust foundation in young engineers.

**🔗 Live Deployment:** [https://ai-robotics-web-smit.vercel.app](https://ai-robotics-web-smit.vercel.app)

## ✨ Core Highlights
* **Premium Glassmorphic UI:** Features deep navy gradient layouts, translucent frosted glass cards, and immersive ambient lighting.
* **Intelligent Scroll Engine:** Powered by a lightweight custom `IntersectionObserver` that tracks viewport intersections, seamlessly triggering 3D floating keyframes and staggered cubic-bezier animations.
* **Zero-Dependency Aesthetic:** Achieved strict modern design best practices entirely through native Variable/Vanilla CSS without bloated dependencies.
* **SEO Optimized:** Structured for clarity and indexing utilizing lightweight HTML hierarchy and responsive container techniques.

## 🧰 Technology Stack
- **Core:** React 19 + Vite for blazing-fast HMR and bundling.
- **Styling:** Vanilla CSS Custom Variables (Dark mode targeted).
- **Deployment Strategy:** Vercel CLI (Continuous deployment enabled).

## 🗂️ Curriculum Modules Addressed
1. **Explore & Build (Month 1):** Breadboarding, Electrical Circuits, LEDs, and Buttons.
2. **Move & Sense (Month 2):** Block Logic Integration, IR/Ultrasonic Sensors, Motor Drivers & Chassis.
3. **Smart & Connected (Month 3):** Bluetooth Communication (HC-05), Mobile Control, and Home Automation.

## 🚀 Getting Started Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/shuja-tech/AI-Robotics-Web-SMIT.git
   ```
2. **Navigate into the directory:**
   ```bash
   cd "AI & Robotics web smit"
   ```
3. **Install the required dependencies:**
   ```bash
   npm install
   ```
4. **Boot up the server:**
   ```bash
   npm run dev
   ```
